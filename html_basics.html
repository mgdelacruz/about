HTML BASICS

Paragraphs

This is a paragraph. HTML paragraphs are defined with the <p> tag


Headings

HTML headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading. <h6> defines the least important heading:

This is HEADING 1

This is HEADING 2

This is HEADING 3

This is HEADING 4

This is HEADING 5
This is HEADING 6

Note: closing paragraphs and headers automatically appends breaks


Links

HTML links are defined with the <a >tag:

This is a link to the website I used as a resource

The link's destination is specified in the href attribute.


Entities

The less than and greater than signs displayed on this page, as well as the ampersands below, used character entities as these are reserved characters in HTML. A character entity looks like this:

&entity_name; OR &#entity_number;

Note: Browsers may not support all entity names, but the support for entity numbers is good.


Images

HTML images are defined with the <img> tag. The source file (src), alternative text (alt), width, and height are provided as attributes.

Note: This is a horizontal rule defined by the <hr>tag

HTML Display

Browsers automatically remove any extra spaces and lines when the page is displayed. Thus, making the use of <br> to start a new lines without starting a new paragraph necessary. On the other hand, in the case that we do need to preserve spaces and line breaks such as in displaying poems, we can make us of <pre> which defines preformatted text. To demonstrate we have this following example: 

        My Bonnie lies over the ocean.
                    
        My Bonnie lies over the sea.
                    
        My Bonnie lies over the ocean.
                    
        Oh, bring back my Bonnie to me.

HTML Styles

The HTML style attribute has the following syntax:

< tagname style = "property:value;" >

Styles can be applied to an entire page or just single elements. It is neater to have a seperate file for styles. Here are some possible display modifications we can do:


apply a background color

change the text color

change the text font

change the text size

set the alignment


HTML Formatting

Commonly used text formatting in HTML:


Bold
Strong (displayed bold by default but can be overwritten using CSS)
Italic
Emphasized text (displayed italicized by default but can be overwritten using CSS)
Small (makes text appear smaller)

Highlights can be done by using the mark tag

Del: del defines test that has been deleted from the document. Usually shown as strikethroughed text

Ins: defines text inserted in a document. Typically displayed as underlined text by default
