HTML BASICS

Paragraphs

This is a paragraph. HTML paragraphs are defined with the <p> tag


Headings

HTML headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading. <h6> defines the least important heading:

This is HEADING 1

This is HEADING 2

This is HEADING 3

This is HEADING 4

This is HEADING 5
This is HEADING 6

Note: closing paragraphs and headers automatically appends breaks


Links

HTML links are defined with the <a >tag:

This is a link to the website I used as a resource

The link's destination is specified in the href attribute.


Entities

The less than and greater than signs displayed on this page, as well as the ampersands below, used character entities as these are reserved characters in HTML. A character entity looks like this:

&entity_name; OR &#entity_number;

Note: Browsers may not support all entity names, but the support for entity numbers is good.


Images

HTML images are defined with the <img> tag. The source file (src), alternative text (alt), width, and height are provided as attributes.
